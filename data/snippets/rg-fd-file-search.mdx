`fd` (a fast, user-friendly find) paired with `rg` (ripgrep) makes fast project searching. Example alias:

```bash
alias srch='fd --type f --hidden --exclude .git . | xargs -r rg --line-number --hidden --no-ignore-vcs --colors "match:fg:yellow"'
# usage: srch TODO
```

This includes hidden files while excluding `.git` and adds colored hit highlighting — great for large repos.
