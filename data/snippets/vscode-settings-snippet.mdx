Drop this into your `settings.json` for a clean, consistent editor setup across devices:

```json
{
  "editor.formatOnSave": true,
  "files.trimTrailingWhitespace": true,
  "editor.tabSize": 2,
  "editor.detectIndentation": false,
  "editor.minimap.enabled": false,
  "workbench.colorTheme": "Default Dark+",
  "git.enableSmartCommit": true
}
```

Pair with extensions like `esbenp.prettier-vscode`, `ms-vscode.vscode-typescript-next`, and `ms-azuretools.vscode-docker`.
