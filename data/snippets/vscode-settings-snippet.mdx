Drop this into your `settings.json` for a clean, consistent editor setup across devices.

If you'd prefer this entry to reflect your *actual* local VS Code settings, open the Command Palette in VS Code and choose "Preferences: Open Settings (JSON)", copy the JSON, and replace the example below with your exported settings.

```json
{
  "github.copilot.nextEditSuggestions.enabled": true,
  "gitlens.ai.model": "vscode",
  "gitlens.ai.vscode.model": "copilot:gpt-4.1",
  "workbench.colorTheme": "Andromeda",
  "workbench.iconTheme": "material-icon-theme",
  "chat.editing.confirmEditRequestRemoval": false,
  "security.workspace.trust.untrustedFiles": "open",
  "editor.minimap.enabled": false,
  "explorer.confirmDelete": false,
  "git.enableSmartCommit": true
}
```

Pair with extensions like `esbenp.prettier-vscode`, `ms-vscode.vscode-typescript-next`, and `ms-azuretools.vscode-docker`.
