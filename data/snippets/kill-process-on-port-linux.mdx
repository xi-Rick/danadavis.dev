Quickly find and terminate the process listening on a port (replace 3000):

```bash
sudo fuser -k 3000/tcp || (lsof -i :3000 -t | xargs -r sudo kill -9)
```

This attempts `fuser` first and falls back to `lsof` where `fuser` isn't available. Use with care.
